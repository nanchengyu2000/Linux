## 关于yum的常见命令

- 搜索包   yum  search 软件包名    

  去yum源去搜索该软件

- 下载包  yum install 软件包名

- 卸载软件  yum remove 软件包名

- 升级软件  yum update 软件名

- 升级操作系统，所有软件和内核   yum upgrade
