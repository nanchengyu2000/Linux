### 磁盘情况-工作常用指令

1. 统计一个文件夹下的文件的个数

   ```
   ll | grep "^-" | wc -l
   ```

2. 统计一个文件夹下的目录的个数

   ```
   ll | grep "^d" | wc -l
   ```

3. 统计一个文件夹下的文件个数，包括子文件夹中的文件

   ```
    ll -R | grep "^-" | wc -l
   ```

4. 统计一个文件夹下的目录，包括子文件夹里面的

   ```
   ll -R | grep "^d" | wc -l
   ```

5. 以树状显示目录结构

   ```
   tree 目录
   ```

   tree需要进行安装 yum  install  tree
